Aucune étape ne demande une seconde main

La plupart des apps de « souris à distance » supposent deux mains stables quelque part dans le déroulé. Celle-ci, jamais — y compris la configuration.

Chaque action se fait d'un seul doigt

Pointeur, clic, clic droit, défilement, saisie, lancement d'une app, changement de fenêtre — chacun est un geste à un doigt ou un seul tap. Il n'y a aucun moment « maintenez maintenant ces deux touches », nulle part.

Des cibles grandes et indulgentes

Les commandes sont dimensionnées pour un pouce, pas pour une chasse au pixel. Si votre prise, votre portée ou votre précision varient d'un jour à l'autre, l'interface ne vous en punit pas.

Tapez à votre rythme

Le clavier Composer vous laisse écrire et modifier le message entier sur votre iPhone ou iPad d'abord, puis l'insérer sur le Mac en une seule fois. Pas de course après un curseur en direct dans le champ de quelqu'un d'autre.

Les combinaisons à deux mains deviennent des taps

Les raccourcis clavier qui demandent normalement les deux mains étalées sur un clavier — copier, coller, changer d'app, faire une capture d'écran — se déclenchent d'un seul tap sur votre téléphone.

D'une seule main de bout en bout — configuration comprise

L'appairage consiste à scanner un code QR sur l'écran de votre Mac. Pas de codes à taper, pas d'adresses IP à saisir, pas de fouille dans les réglages.

Installez l'app Mac gratuite

Un petit compagnon dans la barre des menus. Ouvrez-le une fois, approuvez les permissions macOS standard, et il affiche un code QR.

Pointez votre iPhone ou iPad dessus

La caméra ne sert qu'à lire ce code — scannez-le avec le téléphone dans la main qui est libre.

Voilà toute la configuration

Les appareils se connectent directement via votre réseau local ou le Bluetooth, chiffré. Aucun compte à créer, jamais.

Essayez-le selon vos propres conditions
