Installez l'app Mac gratuite

Téléchargez Control My Mac pour Mac — un compagnon léger dans la barre des menus. Ouvrez-le et il affiche un code QR.

Procurez-vous l'app

Installez Control My Mac sur votre iPhone ou iPad dès son arrivée sur l'App Store — votre Mac vous attendra déjà. Gardez les deux appareils sur le même Wi-Fi, ou utilisez le Bluetooth.

Scannez le code QR

Pointez votre iPhone ou iPad vers le code sur votre Mac. Les appareils s'appairent directement — chiffré, d'appareil à appareil — et vous êtes aux commandes.

L'étape 1 prend 30 secondes.
